Galaxy 설치 방법

이 ZIP 파일 자체는 앱으로 설치할 수 없습니다.
아래 폴더 전체를 HTTPS 웹 호스팅에 올리면 PWA 앱으로 설치할 수 있습니다.

가장 간단한 방법
1. ZIP 압축을 풉니다.
2. Netlify Drop 또는 GitHub Pages 같은 HTTPS 호스팅에 폴더 전체를 업로드합니다.
3. 갤럭시에서 생성된 주소를 Chrome 또는 Samsung Internet으로 엽니다.
4. 브라우저 메뉴에서 '앱 설치' 또는 '홈 화면에 추가'를 누릅니다.
5. Barcelona 2026 아이콘이 앱 서랍/홈 화면에 생기고 독립 창으로 실행됩니다.

관광 일정·이미지·관람 가이드는 최초 실행 후 오프라인 캐시됩니다.
Google Maps 및 외부 공식 사이트는 인터넷 연결이 필요합니다.

V8 사진 원칙
- 기존에 임의 크롭하여 재활용한 사진을 모두 제거했습니다.
- 관광지와 정확히 일치한다고 확인된 Wikimedia Commons 파일만 사용합니다.
- 같은 이미지 URL은 사이트 전체에서 한 번만 사용합니다.
- 교통·숙소 휴식처럼 정확한 대표 사진이 불필요한 일정에는 사진을 넣지 않았습니다.
- 외부 Wikimedia 이미지는 최초 온라인 로딩 후 서비스워커가 캐시합니다.
- 사진 파일 수가 줄어 GitHub 웹 업로드가 쉬워졌습니다.
